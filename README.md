# SolaKa: The First Georgian-Language Solana Academy

SolaKa is a specialized, open-source educational platform designed to onboard the Georgian developer community into the Solana ecosystem. By blending localized technical instruction with on-chain verification, SolaKa bridges the linguistic and technical gap for emerging Web3 talent in the Caucasus region.

## 🚀 Vision
Our mission is to democratize high-level blockchain education by providing a hands-on, multi-language (GE/EN/RU) curriculum that rewards learners with verifiable digital credentials (cNFTs).

## 🛠 Technical Architecture
SolaKa is built on the **Anchor Framework** and utilizes a deterministic validation system to ensure academic integrity.

### Key Features:
* **Localized Curriculum:** 5 structured modules covering everything from Airdrops to Solana Pay.
* **On-Chain Progress Tracking:** User state is managed via Program Derived Addresses (PDAs) for immutable progress logging.
* **Compressed NFT (cNFT) Rewards:** Integration with Metaplex Bubblegum for cost-effective, scalable certification.
* **XOR-Fold Verification:** A unique hashing mechanism to verify off-chain task completion on the Solana ledger.

## 📂 Project Structure
The repository includes:
- `/programs`: Anchor-based Rust smart contracts.
- `/frontend`: Next.js UI with i18n support (GE/EN/RU).
- `/tests`: Integration tests for core instruction logic.
